Gerar SAF para DSpace

Instruções

Para gerar o SAF, faça o upload de um arquivo CSV contendo os metadados dos documentos e dos arquivos PDF correspondentes.

Os arquivos PDF devem ter os nomes correspondentes aos especificados no campo filename do arquivo CSV.

O arquivo CSV deve ser salvo no formato UTF-8, utilizando a vírgula ( , ) como separador. Não utilizar ponto-e-vírgula ( ; ).

Caso algum campo contenha mais de uma informação, coloque todas as informações entre aspas duplas e separe os campos com duplo pipe → || .

Exemplo campo dc.creator: "Primeiro Autor||Segundo Autor||Terceiro Autor"

Caso algum campo contenha vírgula ( , ) em sua descrição, coloque toda a descrição entre aspas duplas.

Exemplo campo dc.description.abstract: "Veja, um exemplo a ser seguido para o campo de abstract."

Após o envio, o sistema processará os arquivos e gerará o SAF necessário para importação no DSpace.
